## Bhanu Teja Portfolio
A personal portfolio showcasing projects, skills, and experience built with HTML, CSS, JavaScript, and jQuery.

## Live Site
- https://bhanutejapusarla-portfolio.netlify.app/

## Tech Stack
- HTML5, CSS3, JavaScript, jQuery
- Particle.js, Typed.js, Tilt.js, ScrollReveal, Font Awesome

## Project Structure
- index.html: main portfolio page
- projects/: projects listing page
- experience/: experience page
- assets/: styles, scripts, and images
- skills.json and projects.json: data sources

## Run Locally
Open index.html directly in a browser, or use a local server (for example VS Code Live Server) for best results.
